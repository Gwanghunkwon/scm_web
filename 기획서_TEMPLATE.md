# SCM 자동 일정 관리 시스템 기획서

## 1. 프로젝트 개요
### 1.1 목표
재고, 발주, 생산, 리드타임, 소비기한 데이터를 통합해 **52주 기준 생산·발주 계획**을 자동 생성하고, SCM 담당자의 To-Do를 캘린더로 제공하는 웹 시스템 구축.

### 1.2 해결하려는 문제
- 재고 부족/과잉 재고
- 발주 시점 판단 오류
- 생산 일정 충돌
- 소비기한 관리 실패
- 수작업 엑셀 운영

### 1.3 해결 방식
데이터 기반 자동 계산으로 아래를 생성:
- 생산 요청 시점
- 발주 시점
- 납품 일정

---

## 2. 핵심 기능 요구사항
### 2.1 통합 SCM 대시보드
한 화면에서 핵심 지표를 SKU별 카드 중심으로 제공.

| 항목 | 설명 |
| --- | --- |
| 현재 재고 | 창고 기준 실시간 재고 |
| 예상 사용량 | 판매 예측 기반 예상 소비 |
| 실제 사용량 | 실제 판매/출고 데이터 |
| 예상 발주량 | 자동 계산 발주 제안 |
| 실제 발주량 | 확정된 발주 데이터 |
| 생산 요청량 | 제품 생산 필요 수량 |
| 리드타임 | 생산/물류/원재료 리드타임 |
| 소비기한 | 유통기한 및 임박 상태 |

화면 특징:
- SKU 카드 UI
- 재고 위험도 표시(부족/과잉)
- 소비기한 임박 강조

### 2.2 52주 SCM 계획 자동 생성
계산식(기본):
`필요 생산량 = 예상 사용량 + 안전재고 - 현재재고`

시점 계산:
- 생산 시작일 = 납품일 - 생산 리드타임 - 물류 리드타임
- 발주일 = 생산 시작일 - 원재료 리드타임

결과:
- 52주 주차별 생산/발주/재고 계획 자동 생성

### 2.3 캘린더 기반 To-Do 자동 생성
업무 이벤트를 자동 생성:
- 원재료 발주
- 생산 시작
- 생산 완료
- 물류 출고

보기 모드:
- 주간 / 월간 / 연간

### 2.4 재고 시뮬레이션
사용자 조정 항목:
- 예상 판매량
- 재고
- 생산량
- 발주량

조정 시 52주 계획 즉시 재계산.

### 2.5 소비기한 관리
- 소비기한 임박 알림
- 폐기 예정 표시
- FIFO(선입선출) 우선 출고 지원

### 2.6 재고 위험 알림
| 유형 | 설명 |
| --- | --- |
| 재고 부족 | 생산/발주 필요 |
| 과잉 재고 | 판매 촉진/계획 조정 필요 |
| 소비기한 임박 | 폐기 위험 |
| 발주 지연 | 생산 차질 위험 |

## 3. 데이터 모델(논리)
### 3.1 제품
| 필드 | 설명 |
| --- | --- |
| product_id | 제품 ID |
| product_name | 제품명 |
| category | 카테고리 |
| shelf_life | 소비기한(일) |
| safety_stock | 안전재고 |
| production_capa | 생산 CAPA(일/주 단위) |

### 3.2 재고
| 필드 | 설명 |
| --- | --- |
| product_id | 제품 ID |
| warehouse_id | 창고 ID |
| quantity | 수량 |
| lot_no | 로트 번호 |
| expiry_date | 소비기한 |

### 3.3 생산 계획
| 필드 | 설명 |
| --- | --- |
| product_id | 제품 ID |
| week_no | 주차 |
| production_qty | 생산량 |
| production_start | 생산 시작일 |
| completion_date | 생산 완료일 |

### 3.4 발주
| 필드 | 설명 |
| --- | --- |
| material_id | 원재료 ID |
| week_no | 주차 |
| order_qty | 발주량 |
| moq | 최소 발주량 |
| order_date | 발주일 |
| arrival_date | 입고일 |

## 4. 자동 계산 엔진 정의
### 입력
- 현재 재고
- 예상 판매량
- 안전재고
- 생산/물류/원재료 리드타임
- 소비기한
- MOQ
- 생산 CAPA

### 출력
- 주차별 필요 생산량
- 주차별 발주량( MOQ 반영 )
- 생산/발주/납품 일정
- 캘린더 To-Do 이벤트

구조:

## 9. 현재 기획에서 반드시 포함할 요소
1) 안전재고(Safety Stock)
- 기준: 평균수요 + 리드타임/수요 변동성 반영

2) MOQ(최소 발주량)
- 계산 발주량이 MOQ 미만이면 MOQ로 올림

3) 생산 CAPA
- 일/주 생산 가능량 제약 반영

4) Lot 관리
- 식품/소비기한 제품 필수

5) 창고별 재고
- 공장/물류센터/매장 단위 재고 가시화
